# LakeVista Real Estate

LakeVista Real Estate is a Flutter-based real estate mobile application that provides a foundation for building responsive property listing and browsing functionality. It showcases essential Flutter UI components and layouts for modern real estate apps.

## Features

- Responsive UI designed with Flutter
- Structured project setup for easy extension
- Starter screens for property browsing and app navigation
- Cross-platform support (Android and iOS)

## Tech Stack

- Flutter
- Dart

## Getting Started
